# 概要
こちらは、メモ機能をもつChrome拡張機能のリポジトリです。

# 使用技術
 - JavaScript
 - HTML
 - CSS
 - BootStrap5
 - jQuery

# 実装機能
 - 文字数カウント
 - テキストファイル出力
 - サイズ変更
 - 2言語対応（英語、日本語）
 - ダークモード
 - 自動保存
 - 初期化
